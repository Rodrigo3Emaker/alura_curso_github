Git Github

Lista de comandos adquiridas no curso “Git e Github: Controlo e compartilhe seu código”. Este curso faz parte da Formação “DevOps”, ministrado pela alura.

  1. git init -inicia o git dentro do repositório atual
  2. git status -informa qual é o status atual dos arquivos.
  3. git add . -seleciona todos arquivos dentro do diretório (que serão commitados)
  4. git commit -m "mensagem" -Faz um commit do diretório, ( -m serve para deixar uma mensagem).
  5. git log -mostra informações sobre os commits
  6. git config --local -configura parametros para cada projeto
  7. git config --global -configura parametros para todos projetos no pc
  8. git config --local user.name -configura o nome do autor
  9. git log --oneline -resume as informações de log em uma linha.
  10. git log -p -informações de mudanças que foram realizadas
  11. https://devhints.io/git-log -site com diversos comandos de log
  12. git init --bare -inicia um repositório puro, ou seja, contém somente as alterações
  13. git remote add localDiretório caminhoDiretório -Adiciona um repositório onde, localDiretório se refere ao nome do diretório, e caminhoDiretório é onde está esse diretório.
  14. git remote -retorna o nome do diretório
  15. git remote -v -retorna o endereço do repositório
  16. git clone caminhoDiretório projeto - clona o conteúdo do repositório que está em caminhoDiretório, e salva em um novo diretório chamado projeto
  17. git push localDiretório master - envia o diretório localDiretório / antes deste comamdo é necessário utlizar o comando
    18. : git remote
  18. git pull localDiretório master
    19. -puxa o diretório com o nome localDiretório para o branch master / pode ser necessário alterara o nome do repositório, para isso basta usar o comando: git remote rename origin localDiretório.
  19. git remote rename origin localDiretório - altera o nome do diretório, de origin para localDiretório

Observações

Para "commitar" é preciso realizar os seguintes passos:
Depois de realizar as alterações desejadas, entrar com os comandos:

  1. git add nomeDoArquivo
  2. git commit -m "descriçãoDaAlteração"
  3. git push localDiretório master

Para baixar as alterações feitas:

  1. git pull nomeDoDiretório master
